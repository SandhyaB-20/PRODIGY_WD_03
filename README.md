# Tic Tac Toe Web Application

This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. It is a two-player game that can be played in the browser.

## Features
- Two-player mode
- Interactive user interface
- Automatic win detection
- Reset game option

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was developed as Task 2 for my internship at Prodigy InfoTech to improve my JavaScript logic and frontend skills.

## How to Play
1. Open the index.html file in a browser.
2. Player X and Player O take turns.
3. The game announces the winner or a draw.
4. Click Reset to play again.
